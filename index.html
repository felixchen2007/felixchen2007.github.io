Projects

Arduino Claw Project

APSC 101 · 2026

Built and programmed an Arduino-controlled claw system using electrical components, wiring, and microcontroller logic. My main focus was connecting components, debugging input/output behavior, and adjusting the code so the system responded reliably.

Arduino · C/C++ · Wiring · Debugging · Prototyping

Battery-Powered Vehicle Electrical System

Kwantlen Science Challenge

Worked on the electrical side of a battery-powered competition vehicle. I connected components, organized wiring, soldered parts together, and helped make sure the vehicle could operate under load.

Although our vehicle did not perform as well as expected, comparing it with stronger designs helped me understand the importance of clean wiring, efficient power delivery, and reliable system integration.

Soldering · Circuits · Batteries · Power Delivery · Troubleshooting

Rainwater Harvesting System Model

APSC 101 · Engineering Design

Developed an engineering model for a remote household rainwater harvesting system. The project considered water demand, rainfall data, pump behavior, stakeholder criteria, and system-level tradeoffs.

Excel · Systems Modeling · Hydraulics · Decision Matrix · Technical Writing
