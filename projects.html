Excel Rainwater Harvesting System

APSC 101 · Engineering Design

Developed an Excel-based engineering model for a rainwater harvesting system designed for a remote two-person household. The model considered household water demand, rainfall data, storage behavior, pump behavior, and system-level tradeoffs.

The project required balancing technical performance with stakeholder needs. I used Excel calculations to compare design choices and support decisions related to water availability, reliability, cost, and maintainability. The system was not judged by one number alone; it depended on how multiple design constraints affected each other.

My work included organizing assumptions, building spreadsheet calculations, interpreting pump and system behavior, and communicating results through visuals and presentation slides. This project helped me practice systems thinking because the strongest design had to balance engineering calculations with real user needs.
